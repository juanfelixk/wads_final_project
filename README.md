# Interactive Essay Grading Application

### Architecture - Modular monolith
Modular monolith is chosen as it offers the simplicity of a monolith architecture, where everything can be deployed at once, but also offers the advantage of microservices where the system is separated into different modules based on a business logic. This separation of concern makes it suitable for long-term maintenance of a simple, small web application.
